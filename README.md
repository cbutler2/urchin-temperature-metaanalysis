# The Thermal Responses of Barren-Forming Urchins: Common Patterns and Knowledge Gaps
Data and code in support of the manuscript.

Citation: Butler, C., Boada J., Schuster, J. M., Brown, C. J., Hurd, C. L., Ling, S., Vergés, A., & Bennett, S. (2026). The Thermal Responses of Barren-Forming Urchins: Common Patterns and Knowledge Gaps. Ecosphere.

Overview of files: UrchinReview_CompiledData_clean.csv = data on thermal performance of 5 sea urchins species (_Arbacia lixula, Centrostephanus rodgersii, Heliocidaris erythrogramma, Paracentrotus lividus, Strongylocentrotus dorebachiensis_), as compiled from the existing literature. 

1_dwnld_gbif_data.R, 2_prepare_occurrence_data.R, 3_prepare_coords_sst_extraction.R, 4_prepare_climatology_occurrences.R, 5_prepare_climatology_metaanalysis.R, 6_perpare_metaanalysis_data.R, 7_model_selection.R, gbif_functions = statistical R code used in the analysis presented.
